# 📐 DXF 도면 편집기 - 사진 삽입 프로그램

캐드 DXF 도면을 웹 브라우저에서 열고 사진을 삽입할 수 있는 웹 애플리케이션입니다.

## 🚀 기능

- ✅ DXF 파일 불러오기 및 렌더링
- ✅ 사진 파일 업로드 및 삽입
- ✅ 이미지 위치 이동 (드래그)
- ✅ 이미지 크기 조절 (모서리 드래그)
- ✅ 확대/축소 (줌)
- ✅ 최종 결과 이미지로 저장

## 📦 사용 방법

### 1. 프로그램 실행

1. `index.html` 파일을 웹 브라우저(Chrome, Edge 등)에서 엽니다
2. 또는 로컬 서버를 실행합니다:

```bash
# Python 3이 설치되어 있는 경우
python -m http.server 8000

# 그 다음 브라우저에서 http://localhost:8000 접속
```

### 2. DXF 도면 불러오기

1. **"📁 DXF 파일 열기"** 버튼을 클릭합니다
2. 컴퓨터에서 DXF 파일을 선택합니다
3. 도면이 자동으로 캔버스에 표시됩니다

### 3. 사진 삽입하기

1. **"🖼️ 사진 추가"** 버튼을 클릭합니다
2. 삽입할 사진 파일을 선택합니다
3. 도면에서 원하는 위치를 클릭하여 사진을 배치합니다

### 4. 이미지 편집

- **이동**: 이미지를 클릭하고 드래그합니다
- **크기 조절**: 이미지 모서리의 파란색 핸들을 드래그합니다
- **확대/축소**: 
  - 마우스 휠을 사용합니다
  - 또는 "🔍 확대/축소" 버튼을 클릭합니다

### 5. 저장하기

1. **"💾 이미지로 저장"** 버튼을 클릭합니다
2. PNG 이미지 파일로 저장됩니다

## 🛠️ 기술 스택

- **HTML5 Canvas**: 도면 및 이미지 렌더링
- **JavaScript (Vanilla)**: 핵심 로직
- **DXF Parser**: DXF 파일 파싱 라이브러리
- **CSS3**: 모던 UI 디자인

## 📋 지원 형식

- **입력 DXF**: AutoCAD DXF 파일 (R12, R14, R2000 등)
- **입력 이미지**: JPG, PNG, GIF, BMP 등 웹 표준 이미지 형식
- **출력**: PNG 이미지

## ⚠️ 알려진 제한사항

- 매우 복잡한 DXF 파일은 렌더링이 느릴 수 있습니다
- 일부 고급 DXF 엔티티는 지원하지 않을 수 있습니다
- 인터넷 연결이 필요합니다 (DXF Parser CDN 사용)
